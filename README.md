# React Promise State

Handling loading, error, and success callback of promise out of the box!

![React native web demo](docs/demo.gif)

## Instalation
````bash
yarn add @traveloka/rps
````

## Run Example
````bash
yarn
yarn bootstrap
````

#### App
````bash
cd examples/apps
yarn start
react-native run-ios # open another terminal
````

#### Web
````bash
cd examples/web
yarn start
````
